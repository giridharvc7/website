# giridharvc7.github.io/website

Personal website — work, writing, and projects.

Built with [Astro](https://astro.build), hosted on [GitHub Pages](https://giridharvc7.github.io/website).

## Commands

| Command | Action |
| :--- | :--- |
| `npm install` | Install dependencies |
| `npm run dev` | Start dev server at `localhost:4321` |
| `npm run build` | Build to `./dist/` |
| `npm run preview` | Preview build locally |

## Content

All content lives in plain files — no CMS, no database.

| What | Where |
| :--- | :--- |
| Homepage bio | `src/content/home/index.md` |
| Work & proof of work | `src/content/work/index.md` |
| Writing posts | `src/content/writing/*.md` |
| Side projects | `src/data/projects.json` |

## Deploy

Pushes to `main` auto-deploy via GitHub Actions to GitHub Pages.
